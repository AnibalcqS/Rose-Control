# Rose Control App 🌹

**Proyecto desarrollado como parte de la asignatura de programación móvil**

---

## 📌 Descripción

Rose Control es una aplicación móvil para **gestión de producción agrícola**, enfocada en el registro de agroquímicos, control de producción y seguimiento de variedades de rosas.  
La aplicación permite:  

- Seleccionar agroquímicos y agregarlos a un carrito de compras.  
- Visualizar y eliminar productos seleccionados en el carrito.  
- Registrar producción y mantener historial de actividades.  
- Acceder a información económica y materiales utilizados.  

La interfaz fue desarrollada utilizando **Android Studio con Jetpack Compose** y se apoyó en **inteligencia artificial** para generar sugerencias en el diseño y estructura de la interfaz.  

---

## 📱 Pantallas principales

1. **Home** – Pantalla principal con acceso a todas las secciones.  
2. **Agroquímicos** – Lista completa de productos agrícolas, con opción de selección.  
3. **Carrito de Compras** – Visualiza los productos seleccionados y permite eliminarlos.  
4. **Registro de Producción** – Permite registrar actividades diarias y producción de cultivos.  
5. **Historial** – Lista de registros anteriores.  
6. **Configuración** – Ajustes de la aplicación.  

---

## ⚙ Tecnologías utilizadas

- **Lenguaje:** Kotlin  
- **IDE:** Android Studio  
- **Arquitectura:** Jetpack Compose  
- **Base de datos:** Room (para almacenar agroquímicos y carrito)  
- **Autenticación y almacenamiento en la nube:** Firebase (Auth y Firestore)  
- **Control de versiones:** Git + GitHub  

---

## 💻 Conceptos de programación aplicados

- **Sentencias de control:** `if`, `when`, `for`, `while`  
- **Funciones:** Creación de funciones para manejar la lógica de selección de productos, eliminación del carrito y navegación.  
- **Programación orientada a objetos (POO):**  
  - Clase `Producto` para definir los agroquímicos  
  - Objetos y listas para manejar los carritos de compras  
  - Reutilización de componentes con Composables en Jetpack Compose
